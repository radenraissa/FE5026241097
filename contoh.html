IS Competition merupakan kompetisi teknologi informasi yang diadakan untuk mendorong inovasi dan kreativitas para developer di Indonesia.

Sebagai ajang untuk mengembangkan bakat dan kemampuan para developer di Indonesia, IS Competition mengadakan 3 bidang lomba, yakni Data Analyst, Software Engineer, dan UI/UX Designer.

IS Competition Categories

1️⃣ Data Analyst Competition

Peserta akan diberikan dataset nyata dari dunia industri untuk dianalisis menggunakan tools seperti Python, SQL, atau software analitik lainnya.

Tujuannya adalah menghasilkan insight bisnis yang dapat membantu perusahaan dalam mengambil keputusan strategis.

2️⃣ Software Engineering Competition

Dalam kategori ini peserta akan mengembangkan sebuah aplikasi digital yang dapat menyelesaikan masalah nyata di masyarakat atau dunia bisnis.

Teknologi yang digunakan dapat berupa Web Development, Mobile Apps, ataupun sistem berbasis cloud.

3️⃣ UI / UX Design Competition

Peserta ditantang untuk merancang pengalaman pengguna yang intuitif dan menarik menggunakan tools seperti Figma atau Adobe XD.

Fokus utama dari kategori ini adalah bagaimana desain dapat meningkatkan kenyamanan dan efektivitas pengguna saat menggunakan sebuah aplikasi digital.
